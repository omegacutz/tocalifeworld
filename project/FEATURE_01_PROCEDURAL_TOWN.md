# Feature 01: Procedural Town A-to-B Prototype

## Objective

Build a simple 2D game where the player starts at A and must reach B through a procedurally generated town.

## Included in this feature

- Random seed-based town generation
- Road path from A to B
- Trees, buildings, and NPC people
- Player movement with arrow keys or WASD
- Win condition when reaching B
- Regenerate town with R key

## Teaching moments for Matthew

- Procedural generation means "the computer creates a new map from rules"
- Game loop: input -> update -> draw
- Random seed lets us create many town versions

## Stretch ideas (not required now)

- NPC speech bubbles
- Collect one item while traveling to B
- Timer challenge mode
