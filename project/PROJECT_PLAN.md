# Project Workspace Plan

## Project

Matthew's Mini Town

## Current feature being implemented

- Feature 01: Procedural Town A-to-B Prototype
  - Generate a small town map from a random seed
  - Place trees, buildings, and people (NPCs)
  - Move player from Start (A) to Goal (B)
  - Win when player reaches Goal

## Next features queue

- Feature 02: Deterministic seed controls
- Feature 03: Simple interactions with NPCs
- Feature 04: Quest markers and checkpoints
- Feature 05: Save/load game state

## Session rule

Each coding session should end with:

1. One playable improvement
2. One note from Matthew about what felt fun
3. One tiny goal for next session
